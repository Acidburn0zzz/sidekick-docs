# Open Existing App

If you already have an existing {{ site.ns }} app, you can open it in {{ site.sk }} and continue your development from there.

## Procedure

1. Open {{ site.ns-sk }}.
1. Click **Open**.
1. Select the folder in which your {{ site.ns }} app is located. Click **Select Folder**.
